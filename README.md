# Weather Forecast Classification

A machine learning project for weather forecast classification using Logistic Regression and Random Forest with preprocessing and hyperparameter optimization pipelines.

## Features
- Weather forecast classification
- Logistic Regression classifier
- Random Forest classifier
- Machine learning pipelines
- Feature scaling and preprocessing
- Feature selection using SelectKBest and SelectPercentile
- Hyperparameter tuning with GridSearchCV
- Stratified K-Fold cross-validation
- Model evaluation using confusion matrix and classification report
- Best model serialization with Pickle

## Files
- `weather_classification_ml_pipeline.ipynb` → Jupyter Notebook
- `best_random_forest.pkl` → Saved trained model
- `dpc.csv` → Weather dataset

## Technologies
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Description
This project explores supervised machine learning techniques for weather forecast classification using preprocessing pipelines, feature selection, Logistic Regression, Random Forest, and hyperparameter optimization.
